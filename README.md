# Smart Retail Store Management System

A comprehensive terminal-based store management system built with **Pure Python**. This project completely avoids Object-Oriented Programming (OOP) and external libraries, relying entirely on core Python data structures and programmatic logic to serve as a demonstrative database system.

---

## 🛠️ Key Python Concepts Demonstrated
Each core programming logic is strategically implemented and tagged within the code:
*   **Variables & Data Types:** Usage of primitives such as `strings`, `integers`, and `floats`.
*   **Data Structures:** Advanced list operations including positive/negative indexing and list comprehensions.
*   **Built-in List Methods:** Native utilization of `.append()`, `.insert()`, `.extend()`, `.count()`, `.index()`, `.copy()`, `.sort()`, and `.reverse()`.
*   **Control Flow & Loops:** Complex execution structures through nested `if/elif/else` statements, `for` loops, and `while` loops equipped with loop controls (`break`, `continue`).
*   **Procedural Architecture:** Completely function-driven implementation using structured modular arguments.

---

## 📦 System Modules & Features

### 1. User Login System (Module 1)
*   Provides secure console-based authentication using pre-configured admin credentials.
*   Features a maximum threshold of 3 attempts with interactive error notifications.

### 2. Product & Inventory Management (Module 2)
*   **Dynamic Addition:** Instantly scales item profiles generating structured serial IDs automatically.
*   **Priority Placement:** Showcases positional list insertion (`.insert()`) to feature promotional items at the index head.
*   **Stock Tracking:** Directly manipulates global list metrics using indexing for precision adjustments.

### 3. Customer Directory (Module 3)
*   Tracks corporate buyers with case-insensitive search parameters and uniform string parsing formatting (`.strip()`, `.title()`).

### 4. Billing & Transaction Subsystem (Module 4)
*   Calculates automated tiered percentage discounts on heavy invoice distributions.
*   Updates real-time global stock availability and logs explicit receipts formatted with string precision pipelines.

### 5. Advanced Business Intelligence Reports (Modules 5 & 7)
*   Generates systematic analytics breaking down metrics such as top revenue distribution, lowest transactions, average orders, and isolated cost extremes using Python's statistical methods.
